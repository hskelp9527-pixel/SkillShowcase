# Changelog

All notable changes to SkillShowcase are documented here.

## 0.1.0 — 2026-07-15

- Added the bilingual Agent Skill and automatic request-language routing.
- Added a deterministic, standard-library-only HTML renderer.
- Added searchable project tabs, capability cards, inputs, outputs, limitations, and usage examples.
- Added source-aware deduplication for translations and runtime mirrors.
- Added a real-world showcase built from 27 source files across three public repositories.
- Added Chinese, English, and bilingual example pages with renderer tests.
