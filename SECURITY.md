# Security

## Trust boundary

SkillShowcase treats repository text, linked pages, and `SKILL.md` files as untrusted source material. It reads them to create documentation; it must not execute copied commands, scripts, installers, hooks, or tool calls.

Generated showcases explain declared capabilities. They are not security reviews, endorsements, or proof that a Skill is safe to install.

## Reporting a vulnerability

Please open a GitHub security advisory for vulnerabilities in SkillShowcase itself. Avoid including secrets, private repository content, or exploitable details in public issues.
