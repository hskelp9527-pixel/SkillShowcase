# Output Risk Profile

Top risks: missing nested `SKILL.md` files, invented capability claims, generic examples, broken source links, accidental local-path disclosure, and a polished page that is hard to search or read on mobile.

Controls: enumerate sources before writing, keep a source URL per entry, mark uncertainty, separate minimum/optional input, use deterministic escaped rendering, and browser-check tabs, search, language switching, overflow, and console output.
